untuk membuka source code:
1. install aplikasi seperti codeblock dulu atau semacamnya yang support untuk C
2. download semua file yang ada
3. buat semua menjadi satu folder
4. buka file-file tadi dengan aplikasi codeblock yang sudah terinstall
5. run pada file main.c agar terlihat hasilnya

Note: kenapa menjadi satu folder, karena main.c memanggil beberapa function yang terletak difile yang lain tadi.
